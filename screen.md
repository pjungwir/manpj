## STARTING

Create a new session named *name*:

```
screen -S *name*
```

## COMMANDS

- ^A'
  list windows of the current session.

- ^A0..9
  switch to window 0..0.

- ^A^A
  switch the last window.

- ^Ac
  create a new window and open it.

- ^A^C
  create a new window and open it.

- ^Aa
  send ^A to the current window.

- ^AA
  title the current window.
